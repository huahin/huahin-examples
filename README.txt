Huahin Examples is an example run using the Huahin Core.
Equipped with the example of several.
Perform the development of applications with reference to this example.

Example that is registered, is an application to manipulate all the Web log.

* Deta Sort
    Get the 10 most recent at the specified URL.

* Path Ranking
    Get the PV and UU of page for Top 50 by date.

* User Ranking
    Get the user ranking of each page for Top 3.

* Huahin Word Count
  Word count for Huahin.

* Natural Word Count
  Word count for natural MapReduce.
